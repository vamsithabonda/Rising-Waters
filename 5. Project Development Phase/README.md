# 🌊 Rising Waters – Flood Prediction Using Machine Learning



## 📌 Project Overview



Rising Waters – Flood Prediction Using Machine Learning is a web-based application that predicts the possibility of floods based on weather and environmental parameters. The project uses machine learning algorithms to analyze the input data and provide accurate flood risk predictions through a simple Flask web application.



---



## 🎯 Objectives



- Predict flood occurrence using machine learning techniques.

- Help users identify flood risk based on weather conditions.

- Provide an easy-to-use web interface for prediction.

- Compare multiple machine learning models and select the best-performing model.



---



## 🚀 Features



- User-friendly web interface

- Weather parameter input

- Instant flood prediction

- Machine Learning-based prediction

- Responsive design

- Fast and accurate results

## ▶️ How to Run the Project



### 1. Clone the Repository



```bash

git clone <https://github.com/vamsithabonda/Rising-Waters.git>

```



### 2. Navigate to the Project Folder



```bash

cd Flood-Prediction-Using-Machine-Learning

```



### 3. Install Dependencies



```bash

pip install -r requirements.txt

```



### 4. Run the Flask Application



```bash

python app.py

```



### 5. Open the Browser



```

http://127.0.0.1:5000

```
